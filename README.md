# CAP's Yeti Tool
CAP's Yeti Tool v1.05 (Autodesk Maya)

Tested on Yeti v3.08 & Maya (2018 & 2019)

How to use:
- Put CAP_Yeti_Tools.mel into C:\Users\YourName\Documents\maya\201#\scripts\
- Open Maya.
- Run MEL Command CAP_Yeti_Tools(); from console.
- Done.
